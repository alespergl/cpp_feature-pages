C++ Feature Support

This page presents the results of running the cpp_feature test suite on various C++ toolchains.

You can drill into each test case by clicking on the respective cell in the grid. There you will see the source code of the test, compiler and linker output, and the output of the built program.

Each test case is designed to test support of one C++ standard feature at a certain level. There is always a header comment which points to the relevant document describing the feature. Also every test case provides information on how to check for the feature in source code when using the cpp_feature library.

Many of the test cases have parts based on examples from the standard or various online sources.
